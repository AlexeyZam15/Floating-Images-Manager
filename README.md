# 🖼️ Floating Image Manager

Просмотр изображений в плавающих окнах поверх всех программ.

## 🚀 Быстрый старт

```bash
pip install pillow
python main.py
```

## ✨ Основные возможности

- **Плавающие окна** — всегда поверх других приложений
- **Умный зум** — масштабирование под курсор (колесо мыши)
- **Панорамирование** — Ctrl + ЛКМ
- **Вставка из буфера** — Ctrl+V
- **Сохранение настроек** — все параметры запоминаются

## ⌨️ Горячие клавиши

| Клавиши | Действие |
|---------|----------|
| `Ctrl+O` | Открыть файлы |
| `Ctrl+V` | Вставить из буфера |
| `Ctrl+A` | Показать все |
| `Ctrl+W` | Закрыть все окна |
| `F1` / `Ctrl+S` | Настройки |
| `Del` | Удалить из списка |
| `Esc` | Закрыть всё |

**В окне картинки:**

- Колесо — зум (Ctrl — медленно, Shift — быстро)
- Ctrl + ЛКМ — панорамирование
- Средняя кнопка — сброс
- ПКМ — меню

## 🎯 Управление размером окна

- **Обычное растягивание** — масштабирует картинку
- **Shift + растягивание** — с сохранением пропорций
- **Ctrl + растягивание** — только размер окна

## ⚙️ Настройки

Сохраняются в: `%LocalAppData%\floating_images\settings.json`

Параметры:

- Скорость зума (медленный/обычный/быстрый)
- Границы зума (мин/макс)
- Задержка скрытия панели
- Размер границ окна

## 📦 Зависимости

- Python 3.7+
- Pillow
