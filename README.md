## 📥 Установка из исходников (для разработчиков)

### Требования
- Python 3.8 или выше
- Библиотека Pillow

```bash
pip install Pillow
```

### Запуск из исходников

```bash
git clone https://github.com/your-username/floating-images-manager.git
cd floating-images-manager
python main.py
```

### Самостоятельная сборка EXE

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name "FloatingImagesManager" main.py
```

Готовый EXE файл появится в папке `dist/`

## 🎮 Использование

### Главное окно галереи

| Действие | Горячая клавиша |
|----------|----------------|
| Открыть изображения | `Ctrl+O` |
| Вставить из буфера обмена | `Ctrl+V` |
| Показать все изображения | `Ctrl+A` |
| Закрыть все плавающие окна | `Ctrl+W` |
| Скрыть/показать все окна | `H` |
| Открыть настройки | `F1` или `Ctrl+S` |
| Удалить выбранные из списка | `Del` |
| Выход из программы | `Ctrl+Q` |

### Плавающее окно с изображением

| Действие | Управление |
|----------|------------|
| Масштабирование (обычное) | Колесо мыши |
| Масштабирование (медленное) | `Ctrl` + Колесо мыши |
| Масштабирование (быстрое) | `Shift` + Колесо мыши |
| Сброс масштаба | Средняя кнопка мыши |
| Панорамирование (при зуме > 100%) | Зажать левую кнопку мыши и двигать |
| Перемещение окна | Зажать левую кнопку на заголовке или на изображении (при зуме 100%) |
| Изменение размера окна | Перетаскивание границ или углов |
| Изменение размера с сохранением пропорций | `Shift` + перетаскивание границы |
| Изменение размера с центрированием изображения | `Ctrl` + перетаскивание границы |
| Контекстное меню | Правая кнопка мыши |
| Закрыть окно | `Esc` или через контекстное меню |

### Контекстное меню плавающего окна

- **Увеличить** — масштабирование с обычной скоростью
- **Уменьшить** — масштабирование с обычной скоростью
- **Сбросить окно** — сброс масштаба и размера окна к исходным значениям
- **Сохранять пропорции** — переключение режима фиксации пропорций для масштабирования
- **Оптимальный размер** — установка предустановленного оптимального размера окна
- **Копировать путь** — копирование полного пути к файлу изображения в буфер обмена
- **Закрыть** — закрытие плавающего окна

## 📂 Структура хранения данных

При первом запуске программа создает следующую структуру в папке `Documents/floating_images`:

```
Documents/floating_images/
├── config/
│   ├── settings.json         # Настройки программы (скорости зума, лимиты, язык и т.д.)
│   └── gallery.json          # Список путей к добавленным изображениям
└── storage/                  # Хранилище изображений из буфера обмена
    └── clipboard_*.png       # Автоматически сохраненные изображения
```

**Важно:**

- Обычные изображения (открытые через кнопку "Открыть") остаются на своих местах и не копируются в папку `storage/`
- Изображения из буфера обмена автоматически сохраняются в `storage/` в формате PNG
- При удалении изображений из галереи файлы из `storage/` удаляются с диска полностью
- При удалении обычных изображений из галереи удаляется только запись, исходные файлы остаются нетронутыми

## 🔧 Настройки программы

Все параметры доступны в окне настроек (открывается по `F1` или через меню):

### Скорость зума

- **Медленный зум** — коэффициент масштабирования при зажатом `Ctrl` (по умолчанию: 1.03)
- **Обычный зум** — коэффициент масштабирования при обычном вращении колеса (по умолчанию: 1.08)
- **Быстрый зум** — коэффициент масштабирования при зажатом `Shift` (по умолчанию: 1.25)

### Ограничения зума

- **Минимальный зум** — наименьший возможный масштаб (по умолчанию: 0.3x)
- **Максимальный зум** — наибольший возможный масштаб (по умолчанию: 10x)

### Интерфейс

- **Задержка скрытия кнопок** — время в миллисекундах, через которое скрываются элементы управления при уходе мыши (по
  умолчанию: 800 мс)
- **Размер границы окна** — толщина области перетаскивания для изменения размера в пикселях (по умолчанию: 8 пикселей)

### Анимация

- **Плавная анимация зума** — включение/отключение плавного масштабирования
- **Длительность анимации** — время анимации в миллисекундах (по умолчанию: 150 мс)

### Язык

- **Русский** — интерфейс на русском языке
- **English** — интерфейс на английском языке (требует перезапуска программы)

### Сброс настроек

Кнопка "Сбросить" восстанавливает все настройки к значениям по умолчанию.
